# 儿童算数小能手 - RPG游戏冒险型

## 项目概述

一个面向1-6年级学生的数学学习RPG游戏，通过游戏化方式提升数学学习兴趣。

## 技术栈

- **前端**: React + TypeScript + Tailwind CSS
- **游戏引擎**: Phaser.js (2D游戏)
- **后端**: Node.js + Express
- **数据库**: SQLite (MVP) → PostgreSQL (生产)

## 快速开始

```bash
npm install
npm run dev
```

## 项目结构

```
src/
├── components/     # UI组件
├── game/          # 游戏逻辑
├── data/          # 题库数据
└── utils/         # 工具函数
```
