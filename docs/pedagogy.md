# 儿童算数小能手 - 教学方法论

## 1. 教育理念

### 1.1 核心理念

**游戏化学习 (Gamification Learning)**

将枯燥的数学练习转化为有趣的RPG冒险体验，通过游戏机制激发儿童的内在学习动机。

### 1.2 理论基础

- **自我决定理论 (SDT)**：满足自主性、胜任感、关联性三大心理需求
- **心流理论 (Flow Theory)**：平衡挑战与技能，保持最佳学习状态
- **间隔重复 (Spaced Repetition)**：科学安排复习频率，强化长期记忆
- **即时反馈 (Immediate Feedback)**：错误即时纠正，正确及时强化

## 2. 学习目标设计

### 2.1 知识目标（按年级分层）

#### 一年级 (Grade 1)
- 20以内加减法
- 简单图形认识
- 100以内数的认识

#### 二年级 (Grade 2)
- 100以内加减法
- 表内乘法
- 简单测量

#### 三年级 (Grade 3)
- 万以内加减法
- 多位数乘一位数
- 分数初步认识

#### 四年级 (Grade 4)
- 多位数乘除法
- 小数认识与运算
- 简易方程

#### 五年级 (Grade 5)
- 分数四则运算
- 小数乘除法
- 简单统计

#### 六年级 (Grade 6)
- 分数、小数、百分数互化
- 比和比例
- 圆的周长和面积

### 2.2 能力目标

- **计算能力**：快速准确进行口算、笔算
- **逻辑思维**：理解数学概念，发现规律
- **问题解决**：运用数学知识解决实际问题
- **学习习惯**：培养持续学习的兴趣和毅力

## 3. 游戏化机制设计

### 3.1 核心循环

```
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│   ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐ │
│   │ 选择关卡 │───▶│ 答题对战 │───▶│ 获得奖励 │───▶│ 角色成长 │ │
│   └─────────┘    └─────────┘    └─────────┘    └─────────┘ │
│        ▲                                            │       │
│        └────────────────────────────────────────────┘       │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

### 3.2 激励机制

| 机制 | 设计 | 教育目的 |
|------|------|----------|
| 经验值 | 答题正确获得经验 | 量化学习进步 |
| 等级 | 经验积累升级 | 阶段性成就感 |
| 装备 | 正确率决定稀有度 | 激励高质量学习 |
| 技能 | 知识点解锁技能 | 学习目标可视化 |
| 成就 | 特殊成就系统 | 多元化激励 |

### 3.3 难度自适应

**动态难度调整算法**：

```
初始难度 = 年级基准难度

每答对一题 → 难度 + Δ
每答错一题 → 难度 - Δ

Δ 根据：
- 连续正确/错误次数
- 答题时间
- 历史正确率
```

### 3.4 挫折保护

- **连续错误提示**：3次错误后提供解题提示
- **新手保护**：前期失败不扣除资源
- **进度保存**：中途退出保留答题进度
- **鼓励机制**：失败后显示进步空间而非单纯分数

## 4. 答题对战设计

### 4.1 对战流程

```
1. 敌人出现 → 显示题目
2. 玩家思考 → 选择/输入答案
3. 判定结果：
   - 正确：对敌人造成伤害
   - 错误：被敌人攻击
4. 技能触发（可选）
5. 循环直到一方HP归零
6. 结算奖励
```

### 4.2 题目呈现原则

- **单题单屏**：避免视觉干扰
- **时间压力适度**：有倒计时但不紧迫
- **即时反馈**：答案正确/错误立即显示
- **解析展示**：错题必展示解析

### 4.3 连击系统

| 连击数 | 奖励加成 | 教育意义 |
|--------|----------|----------|
| 3连击 | 伤害+10% | 鼓励持续专注 |
| 5连击 | 伤害+25% | 激励连贯正确 |
| 10连击 | 伤害+50% | 追求卓越表现 |

## 5. 知识图谱设计

### 5.1 知识点关联

```
加法基础
    ├── 进位加法
    │       └── 多位数加法
    └── 不进位加法
            └── 减法基础
                    ├── 退位减法
                    └── 不退位减法
```

### 5.2 前置关系

学习某知识点前，需先掌握前置知识点：

| 知识点 | 前置知识点 |
|--------|------------|
| 多位数加法 | 进位加法 |
| 表内乘法 | 加法运算 |
| 分数加减 | 分数认识 |

### 5.3 掌握度计算

```
掌握度 = (正确次数 × 权重总和) / 总答题次数

权重：
- 近期答题权重更高
- 连续正确权重更高
```

## 8. 评估与迭代

### 8.1 关键指标

- **DAU/MAU**：日活/月活用户比
- **留存率**：次日、7日、30日留存
- **学习时长**：平均每日学习时间
- **正确率**：整体及分知识点正确率
- **完成率**：关卡/任务完成率

### 8.2 迭代原则

- 数据驱动决策
- A/B 测试新功能
- 用户反馈优先处理
- 教育效果为核心
