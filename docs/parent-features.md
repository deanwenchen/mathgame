# 家长功能设计方案

## 一、功能概述

家长功能旨在帮助家长了解孩子的学习状况，提供适度的监督与引导，同时保护孩子的自主性和学习兴趣。

### 设计原则
1. **透明不监控** - 提供学习数据透明，避免过度监控
2. **引导不控制** - 提供学习建议，尊重孩子自主性
3. **鼓励不施压** - 正向激励为主，避免增加学习压力
4. **隐私保护** - 数据安全存储，家庭隐私保护

## 二、核心功能模块

### 2.1 学习时长统计

#### 数据维度
| 维度 | 说明 | 展示方式 |
|------|------|----------|
| 每日学习时长 | 当天累计学习时间 | 数字 + 进度环 |
| 周学习趋势 | 近7天学习时长 | 折线图 |
| 月度总时长 | 本月累计学习时间 | 数字卡片 |
| 目标达成率 | 每日目标完成情况 | 进度条 + 星星奖励 |

#### 功能设计

**每日学习面板**
```
┌─────────────────────────────────┐
│  今日学习                        │
│  ┌───────┐                      │
│  │  35   │ 分钟                 │
│  │ /60   │ 目标                 │
│  └───────┘                      │
│  ▓▓▓▓▓▓▓▓░░░░░░ 58%            │
│                                 │
│  [详细报告] [设置提醒]            │
└─────────────────────────────────┘
```

**时段控制规则**
- 可设置每日最大学习时长（默认60分钟）
- 可设置允许学习时段（如放学后18:00-21:00）
- 时段外启动需家长授权码
- 连续学习30分钟强制休息提醒

## 三、家长端界面设计

### 3.1 入口设计

**保护孩子体验的设计考量**
- 家长入口独立于游戏主界面
- 进入需要家长验证（密码/指纹/面容）
- 游戏内无家长功能的直接提示
- 孩子视角：看不到家长在"监控"

### 3.2 信息架构

```
家长中心
├── 学习概览
│   ├── 今日学习
│   ├── 本周报告
│   └── 历史记录
├── 学情分析
│   ├── 知识点掌握
│   ├── 薄弱点提醒
│   └── 进步曲线
├── 时长管理
│   ├── 每日限额
│   ├── 学习时段
│   └── 休息提醒
└── 账户设置
    ├── 孩子档案
    ├── 学习目标
    └── 通知偏好
```

## 四、数据统计方案

### 4.1 数据采集

#### 实时采集数据
| 数据项 | 采集时机 | 存储周期 |
|--------|----------|----------|
| 答题记录 | 每次答题 | 永久 |
| 学习时长 | 进入/退出学习 | 永久 |
| 错误记录 | 答错时 | 永久 |
| 休息时间 | 暂停/恢复 | 7天 |

#### 统计计算策略
- **实时统计**：当日数据，实时计算
- **定期聚合**：日/周/月报表，每日凌晨计算
- **缓存策略**：高频查询数据，缓存1小时

## 五、安全控制措施

### 5.1 访问控制

#### 家长验证机制
| 验证方式 | 优先级 | 适用场景 |
|----------|--------|----------|
| 生物识别 | 1 | 支持设备的快捷验证 |
| 数字密码 | 2 | 通用验证方式 |
| 授权码 | 3 | 时段外临时授权 |

#### 权限隔离
```javascript
const permissions = {
  parent: {
    view: ['learning_stats', 'weakness_analysis', 'settings'],
    edit: ['time_limits', 'study_goals', 'notifications'],
    control: ['pause_study', 'extend_time']
  },
  child: {
    view: ['own_progress', 'achievements', 'rewards'],
    edit: ['avatar', 'settings'],  // 仅限游戏内设置
    control: ['start_study', 'pause_study']
  }
};
```

### 5.2 数据安全

#### 存储安全
- 学习数据本地加密存储
- 敏感信息（密码）单向哈希
- 云端备份需家长明确授权
- 数据导出支持（GDPR合规）

#### 传输安全
- HTTPS强制加密
- JWT Token认证
- 请求频率限制（防暴力破解）
- 敏感操作二次验证

### 5.3 隐私保护

#### 数据最小化
- 仅采集必要的学习数据
- 不采集设备无关信息
- 不收集地理位置
- 不进行跨应用追踪

#### 儿童隐私特别保护
- 遵循COPPA（美国）和《儿童个人信息网络保护规定》（中国）
- 家长可随时删除孩子数据
- 学习报告不包含可识别个人身份信息
- 数据共享需家长明确同意
